# Discord-Gemini-Bot
使用Gemini API的大安電研Discord AI聊天機器人

## 如何使用
```pip install -U -r requirements.txt```

將 Gemini API Key 和 Discord Bot Token 放入 `.env.example`

然後將 `.env.example` 更名成 `.env`

運行 `bot.py`
